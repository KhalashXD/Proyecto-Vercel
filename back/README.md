Backend del proyecto

Orden:
- Bot.py: Función para mandar un string a un chat de Telegram.
- Claves.py: Funciones que definen cuantas unidades mandar a una determinada clave.
- Coordenadas.py: Función que determina las coordenadas basándose en las calles.
- Disponibles.py: Función que define los carros disponibles según la DB.
- Orden.py: Función que optimiza el orden de los carros a mandar a un incidente según la disponibilidad y las coordenadas.
- db.py: Funciones para agregar datos a las tablas de la DB.
- Despacho.py: Función que utiliza todas las otras para el despacho inicial de una emergencia.
- Alert.py: Función de tono según compañía.
- Clima.py: Función que extrae datos sobre el clima actual en Villa Alemana y calcula el índice de probabilidad de ocurrencia de incendios forestales.

*Los otros no tienen una funcionalidad específica importante por el momento, pero no borrar*
